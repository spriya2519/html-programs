Paragraphs

the most basic unit of text in an HTML document. Notice that this tag is a container and can contain HTML and other inline HTML elements. This term refers to HTML elements that do not cause a paragraph break but are part of the regular “flow” of the text.

The indenting is optional. Some developers like to use indenting to differentiate a container from its content. It is purely a convention and has no effect on the display of the content.

This element is also a container element and is used to create a logical grouping of content (text and other HTML elements, including containers. This element has no intrinsic presentation. It is frequently used in contemporary CSS-based layouts to mark out sections.
